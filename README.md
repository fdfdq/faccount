# faccount - 智能记账本

## 📊 项目简介

**faccount** 是一款专注于个人财务管理的桌面记账工具，旨在帮助用户轻松记录收支、分析消费习惯、培养良好理财意识。采用本地存储设计，保护用户隐私数据安全。

## ✨ 核心功能

### 🔄 简洁记账
- 快速添加收支记录，支持分类标签
- 收入/支出分类管理
- 支持备注和附件添加

### 📱 我们优秀在哪里？
- 无广告，页面简洁
- 多主题切换，可选择深色模式
- 项目完全开源
